Welcome to PynamoDB's documentation!
====================================

PynamoDB is a Pythonic interface to Amazon's DynamoDB. By using simple, yet powerful abstractions
over the DynamoDB API, PynamoDB allows you to start developing immediately.

Features
========

* Python 3 support
* Python 2 support
* Support for Unicode, Binary, JSON, Number, Set, and UTC Datetime attributes
* Support for DynamoDB Local
* Support for all of the DynamoDB API
* Support for Global and Local Secondary Indexes
* Batch operations with automatic pagination
* Iterators for working with Query and Scan operations
* `Fully tested <https://coveralls.io/r/pynamodb/PynamoDB>`_
